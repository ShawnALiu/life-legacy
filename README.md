# life-legacy (生命·传承)

> “死亡不是生命的终点，遗忘才是。” —— 《寻梦环游记》

## 🌟 项目简介
`life-legacy` 是一个开源的个人数字生命构建框架。它允许你导入自己的人生轨迹（照片、日记、聊天记录、人生大事），训练一个基于大模型的 **数字孪生体**。

- **生前**：它是你的**第二大脑**和**代理人**，帮你处理事务，延续你的工作流。
- **身后**：它是你的**数字墓碑**和**灵魂容器**，让亲人能跨越时空与你对话，聆听你的故事。
